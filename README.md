# 🏥 Healthcare Analytics Dashboard

## Project Overview

This project analyzes 50,000+ hospital records using Power BI, SQL, PostgreSQL, Python, and Excel.

## Tools Used

- Power BI
- SQL
- PostgreSQL
- Python
- Excel

## Dashboard Features

- Total Patients KPI
- Total Revenue
- Average Treatment Cost
- Recovery Rate
- Revenue by Department
- Top Diseases
- Patient Demographics
- Insurance Status
- High-Risk Patient Analysis
